Guitar Harmonics

Posted on Thu 23 June 2022 in Article

Simulating Playing Harmonics

This is an extension of the previous work titled 'Second Simulation: Guitar String'. The idea here is to understand what happens when 'harmonics' are played on a guitar. This is done by lightly holding a finger on a string at exact locations, plucking the string, and removing the finger. The result is a higher pitched sound instead of just plucking and open string.

We can gather what is happening from the mode shapes. The location of where to the finger to generate harmonics is very precise: over the 12th fret (half the length of the string), over the 7th fret (a third of the length), and over the fifth fret (a fourth of the length). If these locations are damping the strings, then the fundamental frequency and the lower overtones are muted.

Any harmonics removes the fundamental mode (mode 1). When harmonics is used halfway on the string, the second mode is allowed to sound since we're at the mode's node with zero velocity. However, the third node is muted, while the fourth is allowed to sound.

When harmoincs is applied one third of the way on the string, the third mode is allowed to sound while mode 2 and mode 1 have velocites there that damp out their sound.

We wish to simulate this event phenomenon.

Modeling

The same model that was developed for the previous string plucking simulation can be used. However, the bond graph is adjusted to include not only damping for each mode shape, but also damping at a location \(x_2\).

The bond graph above shows the resistance \(R\) which will represent a finger touching the string at a location \(x_2\). The resistance is shown as linear (the colon and the resistance coefficient b), but will be applied only for a small amount of time. This makes the system nonlinear.

If one follows the causality once following the procedure, we find that the damping force is applied to each mode momentum transoformed through their appropriate mode shape. The velocity that passes through the damping is dependent on a combination of each the mode shape velocities.

Equations

The equations are not too different. As mentioned in the previous section, the combination of flows from each mode shape needs to be determined. The equations turn out to be as follows:

\(\dot{p}_n = -k_n q_n -b_n \frac{p_n}{m_n} + F_1(t)Y_n(x1) + F_d Y_n(x2)\)

\(\dot{q}_n = \frac{p_n}{m_n}\)

where

\(F_d = -b(\sum_{n=1}^{N} Y_n(x2) \frac{p_n}{m_n})\)
